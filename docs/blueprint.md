# **App Name**: CMCT Compass

## Core Features:

- Informative Content Display: Showcase CMCT's mission, impact, programs, and testimonials across dedicated, beautifully designed sections on the homepage, allowing easy consumption of information.
- Dynamic Image Galleries: Present visual content like activities and projects through modern grid or masonry layouts, offering engaging hover interactions.
- Direct Donation Gateway: Provide a prominent 'Donate Now' Call to Action (CTA) button, facilitating a clear and straightforward path for users to contribute.
- Seamless Navigation: Implement an intuitive and clean navigation system allowing users to effortlessly browse different sections of the website.
- Responsive Design Adaptability: Ensure the website automatically adjusts its layout and functionality for optimal viewing and interaction across all device types, from mobile to desktop (mobile-first approach).
- Interactive Cards and Elements: Engage users with interactive UI elements like program cards and impact statistics, featuring subtle hover effects and visual feedback.

## Style Guidelines:

- The design concept aims to evoke trust, inspiration, and professionalism. The selected palette centers on a light color scheme, anchored by vibrant golden yellows that symbolize hope and energy, complemented by sophisticated neutrals.
- Primary Color: A bright, inviting golden yellow (#FCC91B) serves as the core brand color, embodying optimism and warmth, and draws the eye to key elements. It provides a luminous counterpoint against the clean background.
- Background Color: A pristine white (#FFFFFF) ensures a clean, spacious, and elegant canvas, promoting readability and a premium feel. This stark, bright foundation amplifies the vibrancy of the accent colors.
- Accent Color: A more saturated, lively golden yellow (#FFCC00) is strategically employed for critical Calls to Action and buttons. This shade maximizes visibility and encourages interaction, providing an energetic spark.
- Text Colors: A deep charcoal (#404040) is used for heading text, providing strong contrast and authority, while a balanced medium gray (#8C8C8C) offers elegant readability for secondary text and descriptive content. A dark tone footer uses #404040 for its background.
- Headline font: 'Poppins' (geometric sans-serif) will be used for headlines, conveying a modern, clean, and bold aesthetic suitable for impactful statements. Body font: 'Inter' (grotesque sans-serif) ensures exceptional readability and a neutral, professional tone for all body text.
- Minimal line icons will be utilized throughout the site. These icons are selected for their clarity and modern simplicity, maintaining the clean aesthetic without adding visual clutter, and reinforcing content themes.
- A refined, grid-based layout provides structured alignment and balances visual elements with generous white space. Key sections will employ clean full-width banners, two-column content blocks, and adaptive multi-column card displays (3-4 columns for programs, masonry for galleries) for visual variety and elegance. The design follows a mobile-first philosophy, ensuring clean stacking of sections and optimal performance across all devices.
- Subtle, smooth transitions and micro-interactions will enhance user experience without distraction. This includes soft shadows and gentle rounded corners on UI elements, elegant hover effects for cards and buttons (slight scale and shadow), and smooth zoom on gallery images. Glassmorphism or soft neumorphism will be applied sparingly to create a sense of depth and quality.
